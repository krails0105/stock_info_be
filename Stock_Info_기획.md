# Stock Info

# 📊 주식 초보자용 정보 웹사이트 개발 계획

## ✅ 프로젝트 개요

- **목표**: 주식 초보자를 위한 정보 제공 웹사이트 제작
- **주요 기능**: 섹터별 종목 리스트, PER/PBR, 뉴스, 거래량 등을 알기 쉽게 시각화
- **백엔드**: Spring Boot
- **데이터**: 외부 API 사용 (KIS Developers, Fnguide, Naver 등)
- **프론트엔드**: React 또는 Next.js 기반 설계 예정

## ✅ 현재까지의 진행 현황

### ✔ 기획 확정

- 홈 화면에 **KOSPI/KOSDAQ 지수 + 주식 섹터 리스트** 표시
- 섹터 클릭 시 해당 **섹터의 종목 리스트** 출력
- 섹터는 **관심도 기준**으로 정렬

## ✅ 핵심 기능 (MVP)

### 🏠 홈 화면

- KOSPI / KOSDAQ 지수 실시간 표시
- 관심도 순으로 정렬된 섹터 카드 리스트
- 섹터 클릭 시 해당 섹터 종목 페이지로 이동

### 📊 섹터별 종목 페이지

- 섹터 이름 + 종목 테이블 표시 (종목명, 시가, PER, PBR, 거래량 등)
- 종목 정렬 및 필터 기능 (예: PER 낮은 순)
- 종목 클릭 시 뉴스 링크 / 상세 차트 제공

### 📡 API 목록

| 엔드포인트 | 설명 |
| --- | --- |
| `GET /api/indices` | KOSPI/KOSDAQ 지수 |
| `GET /api/sectors` | 섹터 리스트 (관심도 기준 정렬) |
| `GET /api/stocks?sector=` | 섹터별 종목 리스트 |
| `GET /api/news?code=` | 종목별 뉴스 |

## ✅ 백엔드 설계

- **Spring Boot** + REST API
- 데이터는 외부 API → DTO 변환 → 캐시/DB 저장
- `controller`, `service`, `dto`, `util` 패키지 구조

### 샘플 API 코드

- `SectorController.java`, `SectorService.java`, `SectorDto.java`
- `WebClient` 기반 외부 API 연동 예시 포함

## ✅ 외부 API 후보

| 서비스 | 제공 정보 | 비고 |
| --- | --- | --- |
| KIS Developers | 종목정보, 지수, 시세 | OAuth 필요, 무료 요금제 |
| Fnguide (비공식) | PER/PBR/ROE 등 재무지표 | 크롤링 방식 |
| Naver Finance | 시세, 뉴스, 거래량 | HTML 크롤링 |
| Dart API | 공시/재무제표 | 공식 무료 API |
| Bing News API | 뉴스 자동 수집 | 무료 요금제 있음 |
